# Node.js MongoDB Homework

Bu proje, GoIT Node.js kursunun ikinci ev ödevi için hazırlanmıştır.

## Özellikler

- Express.js ile REST API
- MongoDB Atlas ile veritabanı bağlantısı (Mongoose)
- Tüm iletişimleri listeleme (`GET /contacts`)
- ID ile tek iletişim getirme (`GET /contacts/:contactId`)
- CORS ve logging middleware'leri
- Environment variables ile konfigürasyon

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme modunda çalıştır
npm run dev
```
